#### описание какие файлы будут проигнорированы из файла .gitignore:

> игнорируются все файлы с расширением .tfvars, так как часто являются секретными такими как (пароль, секретные исключения и другие секреты).

> будут игнорироваться файлы override.tf, override.tf.jsom. А так же будут исключатся все файлы  *_ override.tf,  *_override.tf.jsom

> игнорируются все строчки с символом #####

> игнорируются конфигурационные файлы CLI :  .terraformrc,  terraform.rc
